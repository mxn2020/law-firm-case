# Law Firm Case Management System

## Pages & Features

### 1. Dashboard
- Active cases count by status & practice area
- Upcoming deadlines & court dates
- Billable hours this week / month
- Pending invoices & receivables
- Recent activity feed

### 2. Cases / Matters
- Case list with search, status, practice area, attorney filters
- Create new case (client, type, description, opposing party)
- Case detail page: summary, parties, timeline, documents, billing
- Case status workflow (intake → active → pending → closed)
- Related cases linking
- Statute of limitations tracker

### 3. Clients
- Client directory with search
- Client profile: contact info, company, associated cases
- Client intake form
- Conflict of interest check
- Client communication log

### 4. Contacts & Parties
- Contact database (judges, witnesses, opposing counsel, experts)
- Link contacts to cases with role
- Contact detail: organization, phone, email, notes

### 5. Calendar & Deadlines
- Calendar view of court dates, hearings, depositions, meetings
- Deadline tracker with countdown
- Recurring event scheduling
- Court rules-based deadline calculator
- Reminders & notifications

### 6. Documents
- Document repository per case
- Document templates (contracts, motions, letters)
- Version control & audit trail
- Document tagging & categorization
- Secure file sharing with clients

### 7. Time & Billing
- Time entry (manual + timer)
- Billable vs. non-billable classification
- Hourly rate per attorney / matter
- Expense tracking per case
- Invoice generation from time entries
- Trust / retainer account management (IOLTA)
- Payment recording & aging

### 8. Tasks
- Task assignment per case
- Due dates & priority levels
- Checklist templates for case types
- Task completion tracking

### 9. Communications
- Email integration per case
- Secure client portal messaging
- Internal notes & comments
- Call log

### 10. Reports
- Billable hours by attorney, case, period
- Revenue & collections
- Case load by attorney
- Aging receivables
- Productivity metrics
- Export as PDF / CSV

### 11. Settings
- Firm profile & practice areas
- Billing rate configuration
- Court / jurisdiction setup
- Document template management
- User roles & permissions
